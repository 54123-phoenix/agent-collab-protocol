# Contributing

This is an early public RFC. Contributions should focus on making the problem and protocol clearer.

Useful contributions:

- real examples from agent-assisted work
- simpler state card schemas
- alternative item types
- conflict handling proposals
- stale context handling
- integrations with GitHub, Slack, Linear, Jira, Notion, or local files
- critiques explaining why this should not exist

Please prefer concrete examples over abstract arguments.

## Discussion Prompts

- What state should agents be allowed to publish automatically?
- What state should require human review?
- How should conflicting agent claims be displayed?
- Should this be a file protocol, API protocol, browser extension, or app?
- What is the smallest version a real team would use for one week?

## Tone

Be direct, practical, and specific. This project is a shared exploration, not a finished doctrine.
